# get_next_line

## Description

A function in C to take the next line of a file or a file descriptor.

## Requirement

- gcc or Clang

## Usage

```
git clone https://github.com/ryo-manba/get_next_line
cd get_next_line
```
Compile with the main file.

```
gcc -o gnl get_next_line.c get_next_line_utils.c main.c
```

### Now test it with:

```
./gnl test_file.txt
```
